5. Implement `handle(): int`:

```php
public function handle(): int
{
    $iChunk = (int) $this->option('chunk');
    if ($iChunk <= 0) {
        $iChunk = 500;
    }

    try {
        $obService = app(ActiveFlagService::class);
        if (! $obService instanceof ActiveFlagService) {
            $this->error('ActiveFlagService failed to resolve from container.');
            return 1;
        }
        $iReconciled = $obService->reconcileAll($iChunk);
        $this->info(sprintf('Reconciled %d offers (chunk=%d).', $iReconciled, $iChunk));
        return 0;
    } catch (Throwable $obException) {
        $this->error('Recompute failed: ' . $obException->getMessage());
        return 1;
    }
}
```

The instanceof guard exists because Laravel's `app()` helper has a return type that PHPStan L10 can narrow only via instanceof (not via inline @var). The runtime check is essentially free and gives the analyzer the narrowing it needs without inline directives — same pattern as D-03-07-03.

   - Decision link in class docblock: UI-11 / D-29 / D-30 / D-31 / D-32 / D-33.

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-02-01 | Denial of Service | --chunk=0 or --chunk=-1 silently no-ops or loops | mitigate | Coerce non-positive --chunk to default 500 in handle() before passing to service |
| T-04-02-02 | Denial of Service | --chunk=10000000 OOMs on giant catalog | accept | ActiveFlagService::reconcileAll uses chunkById internally — even an over-large chunk just reduces query parallelism, does not OOM beyond that chunk's collection size |
| T-04-02-03 | Tampering | command rewrites operator-managed offers | mitigate | ActiveFlagService::reconcileAll already excludes operator rows at WHERE level (D-03-04-03) AND per-row gate; this command inherits the contract |
| T-04-02-04 | Repudiation | command failure leaves no audit | mitigate | Throwable caught at handle() boundary, written via $this->error(); CLI exit 1 surfaces in any sane orchestration / Forge deploy log |
| T-04-02-05 | Elevation of privilege | command bypasses backend permissions | accept | This is a CLI tool — anyone with shell on the box already has root over the DB. Permission system gates the BACKEND UI (UI-11 covered by reconcileAll's existing operator-skip; the CLI is for ops/admin only) |
