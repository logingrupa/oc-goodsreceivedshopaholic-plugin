<objective>
Build ActiveFlagService — the provenance-aware active-flag reconciler (APPLY-03 / APPLY-04 / QA-05). Reads `SettingsAccessor` (D-01 single accessor, plan 03-01) for the two toggles. For each affected offer, computes the target `active` state based on the 4-cell matrix; SKIPS operator-managed offers entirely. Sets `active_managed_by='plugin'` when toggling so downstream reconciles know the row is plugin-managed.

This service is invoked TWICE in production:
- During Apply (plan 03-07 ApplyOrchestrator wraps it inside the SAME DB transaction as StockApplyService — QA-08 ActiveFlagInsideSameTransactionAsStockApplyTest)
- From the Phase 4 console command (`goodsreceived:recompute_active_from_stock`) — calls `reconcileAll()` chunked

Purpose: Per D-11..D-15, the ENTIRE point of `active_managed_by` is to prevent the plugin from overriding operator decisions. If a QA operator manually deactivates an offer (sets `active=false, active_managed_by='operator'` in the backend), the plugin must NEVER reactivate it on inbound stock. Conversely, when the plugin auto-activates on inbound, it stamps `active_managed_by='plugin'` so a subsequent operator deactivation can RECLAIM ownership.

Output: One service class + two tests covering the 4-cell matrix + the operator-skip case.
</objective>

       <?php
       declare(strict_types=1);
       namespace Logingrupa\GoodsReceivedShopaholic\Classes\Apply;

       use Logingrupa\GoodsReceivedShopaholic\Classes\Support\SettingsAccessor;
       use Lovata\Shopaholic\Models\Offer;
       use October\Rain\Database\Collection as DbCollection;

       /**
        * Provenance-aware active-flag reconciler (APPLY-03 / APPLY-04 / QA-05).
        *
        * Honors:
        *   - active_managed_by='operator' — NEVER touch (operator manually deactivated)
        *   - SettingsAccessor::autoDeactivateOnZero — qty<=0 → active=false
        *   - SettingsAccessor::autoActivateOnStock — qty>0 + previously inactive → active=true
        * When toggling, stamps active_managed_by='plugin' so subsequent reconciles
        * know the row is plugin-managed and re-touchable.
        *
        * Idempotent: a second reconcile with the same offer ids is a no-op (no
        * save fires when offer is already in target state).
        *
        * Writes via saveQuietly — D-15 puts ActiveFlagService inside the SAME
        * DB::transaction as StockApplyService (ApplyOrchestrator plan 03-07);
        * cache flush is centralized in StockApplyService::flushAffectedCaches
        * called by orchestrator AFTER commit.
        */
       final class ActiveFlagService
       {
           private const PROVENANCE_OPERATOR = 'operator';
           private const PROVENANCE_PLUGIN = 'plugin';

           /**
            * Reconcile a specific subset of offers (the typical Apply path).
            *
            * @param list<int> $arOfferIds
            */
           public function reconcile(array $arOfferIds): void
           {
               if ($arOfferIds === []) { return; }

               $bAutoDeactivate = SettingsAccessor::autoDeactivateOnZero();
               $bAutoActivate = SettingsAccessor::autoActivateOnStock();

               // Cheap gate: if both settings are off, no offer can change. Skip query.
               if (! $bAutoDeactivate && ! $bAutoActivate) { return; }

               /** @var DbCollection<int, Offer> $obOffers */
               $obOffers = Offer::whereIn('id', $arOfferIds)->get();

               foreach ($obOffers as $obOffer) {
                   $this->reconcileSingleOffer($obOffer, $bAutoDeactivate, $bAutoActivate);
               }
           }

           /**
            * Reconcile every non-operator offer in chunks. Used by Phase 4 console
            * command (UI-11 goodsreceived:recompute_active_from_stock).
            *
            * @return int  count of offers actually touched (saveQuietly fired)
            */
           public function reconcileAll(int $iChunkSize = 500): int
           {
               $bAutoDeactivate = SettingsAccessor::autoDeactivateOnZero();
               $bAutoActivate = SettingsAccessor::autoActivateOnStock();
               if (! $bAutoDeactivate && ! $bAutoActivate) { return 0; }

               $iTouched = 0;
               Offer::where('active_managed_by', '!=', self::PROVENANCE_OPERATOR)
                   ->chunkById($iChunkSize, function (DbCollection $obChunk) use (&$iTouched, $bAutoDeactivate, $bAutoActivate): void {
                       foreach ($obChunk as $obOffer) {
                           if ($this->reconcileSingleOffer($obOffer, $bAutoDeactivate, $bAutoActivate)) {
                               $iTouched++;
                           }
                       }
                   });
               return $iTouched;
           }

           /** Returns true if a save occurred. */
           private function reconcileSingleOffer(Offer $obOffer, bool $bAutoDeactivate, bool $bAutoActivate): bool
           {
               // Provenance gate (APPLY-03).
               if (((string) $obOffer->active_managed_by) === self::PROVENANCE_OPERATOR) {
                   return false;
               }

               $bTarget = $this->decideTargetState($obOffer, $bAutoDeactivate, $bAutoActivate);
               if ($bTarget === null) { return false; }

               // Idempotent: already in target state → no save.
               if ((bool) $obOffer->active === $bTarget) {
                   return false;
               }

               $obOffer->active = $bTarget;
               $obOffer->active_managed_by = self::PROVENANCE_PLUGIN;
               $obOffer->saveQuietly();
               return true;
           }

           /**
            * 4-cell decision matrix (D-13). Returns null when no change should happen.
            */
           private function decideTargetState(Offer $obOffer, bool $bAutoDeactivate, bool $bAutoActivate): ?bool
           {
               $iQty = (int) $obOffer->quantity;

               if ($iQty <= 0 && $bAutoDeactivate) { return false; }
               if ($iQty > 0 && $bAutoActivate && (bool) $obOffer->active === false) { return true; }

               return null;
           }
       }
