    1. `parse(string $sHtml, string $sSourceFilename): ParsedInvoice` — orchestrator (~25 lines):
       a. `$sStripped = $this->stripBom($sHtml);`
       b. Resolve invoice header BEFORE loading DOM (resolver works on raw HTML; cheaper to fail fast on missing-number than to also build a DOM): `$arHeader = InvoiceNumberResolver::resolve($sStripped, $sSourceFilename);` — bubbles InvoiceNumberMissingException
       c. `$obDom = $this->loadDom($sStripped);` — throws MalformedHtmException on libxml fatal
       d. `$arResult = $this->extractRows($obDom);` — returns `['lines' => list<ParsedLine>, 'skipped' => list<array>]`
       e. If `$arResult['lines']` is empty AND `$arResult['skipped']` is empty: throw MalformedHtmException with arContext `['reason' => 'no_rows_extracted', 'source_filename' => basename($sSourceFilename)]`
       f. `return new ParsedInvoice(invoice_number: $arHeader['invoice_number'], country_code: $arHeader['country_code'], invoice_date: $arHeader['invoice_date'], source_filename: basename($sSourceFilename), lines: $arResult['lines'], skipped_rows: $arResult['skipped']);`

    2. `stripBom(string $sHtml): string` — 3 lines:
       `return ltrim($sHtml, "\xEF\xBB\xBF");` (the BOM bytes; ltrim accepts a charlist, will strip the 3-byte sequence even though `ltrim` is byte-oriented — verified: yes, ltrim removes any character in the charlist set, so all three bytes get stripped).
       Actually: `ltrim("\xEF\xBB\xBF<HTML>", "\xEF\xBB\xBF")` → `'<HTML>'`. Confirmed.

    3. `loadDom(string $sHtml): \DOMDocument` (~12 lines):
       a. `$obDom = new \DOMDocument();`
       b. `libxml_use_internal_errors(true);`
       c. `libxml_clear_errors();`
       d. `$bLoaded = $obDom->loadHTML($sHtml, LIBXML_NONET | LIBXML_NOERROR | LIBXML_NOWARNING);` — `LIBXML_NONET` disables external entity loading (XXE defense, T-02-05-01). `LIBXML_NOERROR | LIBXML_NOWARNING` suppresses ad-hoc warnings; we collect errors below explicitly.
       e. `$arErrors = libxml_get_errors(); libxml_clear_errors();`
       f. Filter errors to FATAL (`level === LIBXML_ERR_FATAL`). If any fatal OR `$bLoaded === false`: throw MalformedHtmException with arContext `['libxml_errors' => array_map(fn($e) => $e->message, $arFatalErrors)]`.
       g. return `$obDom`.

    4. `extractRows(\DOMDocument $obDom): array` (~30 lines):
       a. `$obXpath = new \DOMXPath($obDom);`
       b. `$obRows = $obXpath->query(self::ROW_XPATH);`
       c. `if ($obRows === false) return ['lines' => [], 'skipped' => []];`
       d. `$arLines = []; $arSkipped = [];`
       e. `$iRowCount = 0;`
       f. `foreach ($obRows as $obRow) { if (++$iRowCount > self::MAX_ROWS) break; $arOutcome = $this->parseOneRow($obRow, $iRowCount); if ($arOutcome['line'] !== null) { $arLines[] = $arOutcome['line']; } elseif ($arOutcome['skip'] !== null) { $arSkipped[] = $arOutcome['skip']; } }`
       g. return `['lines' => $arLines, 'skipped' => $arSkipped]`
       PHPDoc: `@return array{lines: list<ParsedLine>, skipped: list<array{row_index:int,reason:string,raw:string}>}`

    5. `parseOneRow(\DOMNode $obRow, int $iRowIndex): array` (~30 lines):
       Returns `['line' => ParsedLine|null, 'skip' => array|null]`. Exactly one of the two is non-null.

       a. Collect TD children:
          ```
          $arTds = [];
          foreach ($obRow->childNodes as $obChild) {
              if ($obChild instanceof \DOMElement && strtolower($obChild->nodeName) === 'td') {
                  $arTds[] = trim($obChild->textContent);
              }
          }
          ```
       b. If `count($arTds) < self::MIN_TD_COUNT`: return `['line' => null, 'skip' => ['row_index' => $iRowIndex, 'reason' => 'insufficient_columns', 'raw' => $this->dumpRow($obRow)]]`
       c. Extract by position:
          - `$sEan = $arTds[2]`
          - `$sName = $arTds[3]`
          - `$sUnit = $arTds[4]`
          - `$sQtyRaw = $arTds[5]`
          - `$sUnitPrice = $arTds[6] ?? null`
          - `$sDiscount = $arTds[7] ?? null`
          - `$sLinePrice = $arTds[8] ?? null`
          - `$sTotal = $arTds[9] ?? null`
       d. Validate EAN: `if (preg_match(self::EAN_REGEX, $sEan) !== 1) { \Log::warning('logingrupa.goodsreceivedshopaholic: row skipped — invalid EAN', ['row_index' => $iRowIndex, 'ean' => $sEan]); return ['line' => null, 'skip' => ['row_index' => $iRowIndex, 'reason' => 'invalid_ean', 'raw' => $sEan]]; }` — D-16 lenient: does NOT throw.
       e. Parse qty (THROWS on bad input — by design, propagates):
          `$iQty = QuantityNormalizer::parseQuantity($sQtyRaw, ['row_index' => $iRowIndex, 'ean' => $sEan]);`
       f. Build ParsedLine:
          ```
          $obLine = new ParsedLine(
              row_index: $iRowIndex,
              ean: $sEan,
              product_name_raw: $sName,
              unit: $sUnit,
              qty: $iQty,
              unit_price: PriceNormalizer::parsePrice($sUnitPrice),
              discount: PriceNormalizer::parsePrice($sDiscount),
              line_price: PriceNormalizer::parsePrice($sLinePrice),
              total: PriceNormalizer::parsePrice($sTotal),
          );
          ```
       g. return `['line' => $obLine, 'skip' => null]`

    6. `dumpRow(\DOMNode $obRow): string` — 5 lines, returns `$obRow->ownerDocument->saveHTML($obRow) ?: ''` (truncate to first 200 chars to keep skip log manageable). Used only for the `raw` field of the skipped_rows record.
